## 🚀 Deploy to Production

### Step 1: Build
```bash

```

### Step 2: Set Environment
Create `.env.local`:
```env
DATABASE_URL=your_database_url
NEXT_PUBLIC_DEMO_EMAIL=demo@example.com
NEXT_PUBLIC_DEMO_PASSWORD=Demo@123456
```

### Step 3: Start
```bash
npm start
```
