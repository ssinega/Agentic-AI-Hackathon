## 📌 Overview

**DiscoveryOS** is an AI-powered Product Discovery & User Research Intelligence platform that helps Product Managers, UX Researchers, and Product Teams analyze customer research at scale.

Instead of manually reviewing interviews, surveys, support tickets, and feedback documents, DiscoveryOS automatically extracts valuable insights, identifies recurring pain points, generates customer personas, clusters themes, prioritizes product opportunities, and creates executive-ready reports.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/ssinega/Agentic-AI-Hackathon.git
cd Agentic-AI-Hackathon
```

---

## Install Dependencies

```bash
pnpm install
```

or

```bash
npm install
```

---

## Configure Environment

Create a `.env.local` file:

```env
DATABASE_URL=

NEXT_PUBLIC_SUPABASE_URL=

NEXT_PUBLIC_SUPABASE_ANON_KEY=

OPENAI_API_KEY=
```

---

## Run Development Server

```bash
pnpm dev
```

or

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
